2. How to use¶

Sphinxの使い方でこれはというものを都度書いていく。

2.1. autobuild¶

make htmlをして更新されたhtmlをブラウザで都度参照というのは手間。 autobuild <https://pypi.python.org/pypi/sphinx-autobuild>`_ を利用すれば、編集したこと自動で検知して、手動時の手間を省いてくれる。:

pip install sphinx-autobuild

autobuldを起動しておくと、以降はファイルを更新して保存すると以下のように自動でビルドが実行される。ファイルを編集してブラウザで実際の表示を見るってシンプルなプロセスになるわけ。
